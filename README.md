# CordelFlix

CordelFlix est une application mobile développée avec **React Native** permettant d’explorer un catalogue de films via l’API **TMDB**.  
L’application offre une navigation fluide entre plusieurs écrans et permet de consulter les détails des films ainsi que d’effectuer des recherches.

---

## Fonctionnalités

- Parcours d’un catalogue de films
- Recherche de films par titre
- Consultation des détails d’un film (description, note, etc.)
- Navigation entre plusieurs écrans
- Récupération des données via une API externe (TMDB)

---

## Technologies utilisées

- React Native
- JavaScript
- API TMDB
